# Leitor de consumo(WATER - GAS)

Um leitor de consumo de agua e gas,integrando,com extração de dados atraves de IA.


## Technologies
 - Typescript
 - NodeJS
 - NestJS
 - MYSQL
 - Swagger
 - Vision(Google Gemini AI)

## To run
`
 docker compose up --build
`
